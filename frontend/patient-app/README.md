# Patient App (Vite React)

患者端前端（手機框 UI）。

AI Studio 原始連結：<https://ai.studio/apps/drive/1Zd8A7FlqHRd1TT9gCL5FHLwNUoCMXfUt>

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies: `npm install`
2. (Optional) Backend base URL: copy `.env.example` to `.env.local` and set `VITE_API_BASE=http://localhost:3001`
3. Run the app: `npm run dev`
